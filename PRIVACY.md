# Privacy Policy

MFCMouseEffect is a local desktop application.

The application does not automatically collect, transmit, sell, or share personal data with the project maintainers or third parties.

The application may process local mouse, keyboard, window, and display information only on the user's device to provide cursor effects, keyboard/mouse indicators, automation mapping, WebSettings, and related desktop features. This data is processed locally and is not uploaded by the application.

Configuration data is stored locally on the user's device.

The WebSettings interface is served locally through a loopback address such as `127.0.0.1` and is intended for local control of the application.

MFCMouseEffect does not include telemetry, advertising analytics, automatic crash report upload, or account tracking.

If future versions add any feature that sends data to the maintainers or a third-party service, this policy will be updated before that feature is released.
