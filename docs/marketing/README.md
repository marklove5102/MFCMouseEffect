## Usage
1. Pick one target subreddit and one clear angle.
2. Use one title from the title bank.
3. Use one matching body template.
4. Add one real screenshot or short GIF before posting.
5. Pre-write one top comment with repo link, platform status, and what feedback you want.

## Guardrails
- Prefer honest engineering language over hype.
- Do not claim full parity when the repo docs only say partial/ongoing parity.
- On Reddit, ask for feedback on one concrete thing instead of asking for generic stars.
